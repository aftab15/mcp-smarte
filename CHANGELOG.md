# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-10-24

### Added
- Initial release of SMARTe MCP Server
- Advanced search tool for finding leads
- Enrich lead tool for enriching contact information
- Enrich account tool for enriching company information
- Weather tools (demo functionality)
- Authentication middleware with Bearer token and API key support
- Request context management for header forwarding
- HTTP service for making API requests
- Health check endpoint
- Comprehensive error handling
- TypeScript type definitions
- Configuration management
- Logging utility
- API documentation
- Contributing guidelines
- README with setup instructions

### Security
- Added authentication gate for all MCP requests
- Environment variable support for sensitive data
- Security headers (X-Content-Type-Options, X-Frame-Options, etc.)
- CORS configuration

## [Unreleased]

### Planned

- Request caching
- Enhanced error messages
- Additional search filters
- Bulk enrichment support
- Webhook support
- Admin dashboard
